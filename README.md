## 概述

三路独立输出，一路USBA-5V3A，两路开关编程可调电压输出。  
可满足带动树莓派的同时提供两路独立控制的3V3V、5V输出。实际使用可根据需求调整。

## 已知问题 / 注意事项

- 两路可调匹配输出的电压并非标准的3V3/5V/9V/10.5V，均有一定的偏差，后期更改可以预留额外电阻位，通过串并联，实现更精确的电压调节。
